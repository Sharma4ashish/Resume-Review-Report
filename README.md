🛠️ Tech Stack

Frontend: React.js / Next.js, Tailwind CSS
Backend: Node.js, Express / Fastify
Database: MongoDB
AI Integration: OpenAI API / LLM APIs
Other Tools: JWT Auth, REST APIs, Cloud Deployment

🎯 Purpose

This project is built to simulate a real-world recruitment system by combining Full Stack Development with Generative AI. It helps users understand how AI can enhance hiring workflows, from resume screening to interview preparation.
